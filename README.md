# Up and Running

## Front-end
### Dependencies
- npm (i use 9.5.1)
- node (i have 18.16.0)
- vue (i have 5.0.8)

### How to run
> $ cd frontend
> 
> $ npm install
> 
> $ npm run serve
> 
> Open http://localhost:8080/
